## 01.운동테이블

```sql
CREATE TABLE exercise(
	year date,
  name varchar(20),
  minute numeric,
  distance numeric,
  calories numeric
)
```

- date
  - 운동한날짜
- name 
  - 운동이름
- minute
  - 운동한 시간 분단위
- distance
  - 런닝또는 걷는것 거리
- calories
  - 칼로리 나오는 운동에 대해서 칼로리
