#### 5.1터미널 이름이 zsh인경우

## 6.나머지 필요한것

- 크롬없으면 설치해야함

- 윈도우는 powershell, cmd

- 맥북은 terminal

  - 신형의 경우 에러가 난다면

    ```shell
    softwareupdate --install-rosetta
    ```

![image-20211226231458920](2021.12.25_0.플러터설치.assets/image-20211226231458920.png)

- 윈도우 경우

  ```shell
  flutter doctor
  ```

  - 개발에 필요한것이 제대로 설치되어 있는지 알려줌

- ! 노란색의 경우 저대로 입력해주고 y 해주면됨

  ```shell
  flutter doctor --android-licenses
  ```
